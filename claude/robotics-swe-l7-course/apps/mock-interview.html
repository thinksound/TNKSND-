📋 How to use (Day 13 — mock day)使い方（13 日目 — 模擬面接の日）

Treat this like the real loop. Pick a round, hit Start on a random prompt, and talk out loud the entire time as if an interviewer is in the room — record yourself (phone/screen-recorder) so you can rewatch. Use the scratchpad only for the notes you'd actually write on the whiteboard; do the thinking verbally. When the timer ends, stop, then score yourself honestly against the E7 rubric, log the session, and write down the single top fix to drill next.
