1 · Quick start

Open mvdr-vs-conventional.html in a browser (double-click — no install; it pulls Tailwind and Chart.js from a CDN, so keep an internet connection the first time). Drag any slider and both beam patterns redraw live. Try the presets (1 Jammer / 2 Jammers / White-noise limit) and watch the MVDR pattern carve a deep null exactly onto each interferer while the Conventional pattern stays fixed.

2 · Conventional vs adaptive

A uniform linear array (ULA) of M mics listens toward a look direction θ₀. A beamformer is just a set of complex weights applied to the mics before summing.

  • Conventional (delay-and-sum) — weights point at θ₀ and never change. Fixed mainlobe, fixed ~−13 dB sidelobes. Data-independent: it doesn't know where interference is, so it can't reject it.
  • MVDR / Capon (adaptive) — minimises total output power while keeping unity gain on θ₀, so it steers deep nulls onto interferers automatically. Data-dependent (uses the spatial covariance R) and optimal in SINR.

3 · The math

Narrowband ULA steering vector for direction θ (c = 343 m/s):

aₘ(θ) = exp( j · m · (2π f d / c) · cosθ ), m = 0 … M−1

Spatial covariance = sensor noise + interferers (power pᵢ from angle θᵢ):

R = σ²I + Σᵢ pᵢ · a(θᵢ) a(θᵢ)ᴴ

Conventional weights (fixed):

w = a(θ₀) / M

MVDR weights — minimise wᴴRw subject to wᴴa(θ₀) = 1:

w = R⁻¹ a(θ₀) / ( a(θ₀)ᴴ R⁻¹ a(θ₀) )

Key fact: set R = σ²I (white noise / large diagonal loading) and MVDR collapses exactly to w = a(θ₀)/M — the conventional beamformer. So conventional is MVDR's special case when nothing is worth nulling.

6 · Metrics & insight

  • White-noise gain (WNG) — robustness to uncorrelated sensor noise (higher = better). MVDR's aggressive nulling can lower WNG; that's the cost of adaptivity, mitigated by diagonal loading.
  • Attenuation @ interferer — the response (dB) in each interferer's direction (lower = deeper null). Compare Conventional vs MVDR to see the adaptive advantage.
  • Insight line — a live sentence summarising what MVDR is doing right now (matching conventional, collapsing under loading, or placing deep adaptive nulls).

7 · Guided experiments

  1. Watch a null form: load 1 Jammer, then drag the interferer angle. MVDR's null tracks it; the conventional sidelobes don't move.
  2. The robustness trade: with a strong jammer, raise Sensor noise / diagonal loading. The MVDR null fills in but WNG improves — directivity vs robustness.
  3. Degrees of freedom: enable both interferers, then reduce M. With too few mics MVDR can't null everything and protect the target at once.
  4. Collapse to conventional: load White-noise limit — the two patterns nearly coincide, demonstrating the special-case relationship.
  5. Resolution: put an interferer close to the look direction and increase M (or spacing) — more aperture lets MVDR separate them.

8 · Limitations & notes

  • Narrowband, far-field, single snapshot — R is built from known angles/powers, not estimated from finite data (no sample-covariance error).
  • Ideal omni elements on a line; no mutual coupling, calibration error, or steering-vector mismatch (real MVDR is sensitive to these — hence loading).
  • Patterns normalized to 0 dB at the look direction; the polar floor is −50 dB.
  • ULA → azimuth only, with front/back mirror symmetry.

MVDR = Minimum Variance Distortionless Response (Capon). c = 343 m/s.

2 · 従来型 vs 適応型

M 本のマイクからなる等間隔直線アレイ（ULA）がルック方向 θ₀ を向きます。ビームフォーマとは、加算前に各マイクへ掛ける複素重みの組です。

  • 従来型（遅延和） — 重みは θ₀ を向き、決して変化しません。固定のメインローブ、固定の約 −13 dB サイドローブ。 データ非依存で、干渉の位置を知らないため抑圧できません。
  • MVDR / Capon（適応型） — θ₀ に利得1を保ちつつ出力総電力を最小化し、干渉源に自動で深いヌルを向けます。 データ依存（空間共分散 R を使用）で、SINR の意味で最適です。

3 · 数式

方向 θ の狭帯域 ULA ステアリングベクトル（c = 343 m/s）：

aₘ(θ) = exp( j · m · (2π f d / c) · cosθ ), m = 0 … M−1

空間共分散 = センサ雑音 + 干渉源（角度 θᵢ・電力 pᵢ）：

R = σ²I + Σᵢ pᵢ · a(θᵢ) a(θᵢ)ᴴ

従来型の重み（固定）：

w = a(θ₀) / M

MVDR の重み — wᴴa(θ₀)=1 のもとで wᴴRw を最小化：

w = R⁻¹ a(θ₀) / ( a(θ₀)ᴴ R⁻¹ a(θ₀) )

重要：R = σ²I（白色雑音／大きな対角ローディング）とすると MVDR はちょうど w = a(θ₀)/M（従来型）に一致します。つまり従来型は、打ち消すべきものが無いときの MVDR の特別な場合です。

5 · プロットとマーカー

極座標ビームパターン

方向に対する応答（dB、外周0から中心−50）。緑=ルック方向、 赤の破線=干渉源。ULA パターンはアレイ軸に対して鏡映対称なので両側を描画。

直交ビームパターン

同じ応答を 0–180°（一意な半分）で表示。ヌルの深さを正確に読むのに最適。縦の破線がルック（緑）と干渉源（赤）を示す。

6 · 指標と解説

  • 白色雑音利得 (WNG) — 無相関なセンサ雑音への頑健性（高いほど良い）。MVDR の積極的なヌル形成は WNG を下げ得る — これが適応性の代償で、対角ローディングで緩和。
  • 干渉源での減衰 — 各干渉源方向の応答（dB、低いほど深いヌル）。従来型と MVDR を比較して適応の利点を見る。
  • 解説行 — MVDR が今何をしているかを要約するライブ文（従来型と一致／ローディングで縮退／深い適応ヌルを形成）。

7 · 実験ガイド

  1. ヌルが形成される様子：妨害1を読み込み、干渉源の角度をドラッグ。MVDR のヌルが追従し、従来型のサイドローブは動かない。
  2. 頑健性のトレードオフ：強い妨害がある状態でセンサ雑音 / 対角ローディングを上げる。MVDR のヌルは埋まるが WNG は改善 — 指向性 対 頑健性。
  3. 自由度：両方の干渉源を有効にし、M を減らす。マイクが少なすぎると、MVDR は全てをヌルしつつターゲットを守れない。
  4. 従来型への縮退：白色雑音の極限を読み込むと2つのパターンがほぼ一致 — 特別な場合の関係を示す。
  5. 分解能：干渉源をルック方向の近くに置き、M（または間隔）を増やす — 開口が広いほど MVDR は両者を分離できる。

8 · 制限事項

  • 狭帯域・遠方界・単一スナップショット — R は既知の角度/電力から構成し、有限データから推定しない（標本共分散誤差なし）。
  • 直線上の理想無指向素子。相互結合・較正誤差・ステアリングベクトル不整合なし（実際の MVDR はこれらに敏感 — ゆえにローディング）。
  • パターンはルック方向で 0 dB に正規化、極座標の下限は −50 dB。
  • ULA → 方位角のみ、前後の鏡映対称あり。

MVDR = 最小分散無歪応答（Capon）。c = 343 m/s。
