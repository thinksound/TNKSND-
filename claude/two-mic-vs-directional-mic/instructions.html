1 · What's being compared

Two ways to get a directional microphone with a first-order pattern B(θ) = α + (1−α)·cosθ (cardioid, hypercardioid, etc.):

  • Single directional element — one capsule whose pattern is formed acoustically (a pressure-gradient capsule with ports/labyrinth). No electronics create the directivity.
  • Two-mic differential array — two omni capsules a small distance d apart, subtracted (with a delay/EQ) to synthesize the same pattern electronically.

They can produce the identical polar pattern. The interesting question is what each one costs — and that's where they diverge sharply.

2 · Directivity — essentially a tie

Directivity Index (gain against diffuse/isotropic noise) for a first-order pattern is a closed form:

DI = 1 / ( α² + (1−α)²/3 ) → cardioid 4.8 dB · supercardioid ≈5.7 dB · hypercardioid 6.0 dB

Both approaches realize the same pattern, so they have the same DI. A first-order design — single element or two-mic — tops out around 6 dB. To go beyond you need a higher-order array (3+ mics), which makes every problem below worse.

3 · White-noise gain — the crux

White-noise gain (WNG) is the array's gain against uncorrelated noise — sensor self-noise, electronic noise, wind turbulence. It tells you how much the processing amplifies the noise that is independent in each channel.

  • Single element: it's one sensor — there is nothing to amplify. WNG ≈ 0 dB, flat with frequency.
  • Differential array: at low frequency the two closely-spaced mics see almost the same signal, so the difference is tiny (∝ kd = ωd/c). To restore unity on-axis response the array multiplies by ~1/(kd), which multiplies the uncorrelated noise too. So WNG falls off a cliff toward low frequency, ≈ +6 dB/octave, and gets worse with smaller spacing:
WNG ≈ 2 / ( α² + 4(1−α)²/(kd)² ) (small-spacing, first order; valid only for kd ≲ 1)

Concretely (cardioid, d = 20 mm): WNG ≈ −0.2 dB at 2 kHz, −5.9 dB at 1 kHz, −19.7 dB at 200 Hz, −25.7 dB at 100 Hz. Halve the spacing and you lose another ~6 dB. This is the single most important practical difference — drag the frequency and spacing sliders and watch the WNG-vs-frequency plot.

Validity: this is the small-spacing approximation. It only holds for kd ≲ 1 (below ≈ c/2πd, ~2.7 kHz at 20 mm). Above that the two mics approach a half-wavelength apart and the array spatially aliases — WNG cannot keep rising and a two-element array can't exceed the ~+3 dB two-sensor ceiling. The simulator uses an exact, bounded two-element model so the high-frequency curve stays physical.

4 · Sensor self-noise floor

WNG turns directly into audible hiss. The effective output self-noise is the capsule's noise (EIN) minus the WNG (in dB):

self-noise_out (dB) = EIN − WNG_dB

Since the array's WNG is negative, its noise floor rises. With 28 dB(A) capsules and 20 mm spacing, the differential cardioid's effective self-noise is ≈ 34 dB(A) at 1 kHz but ≈ 48 dB(A) at 200 Hz and ≈ 54 dB(A) at 100 Hz — that's 20+ dB of extra hiss exactly where rooms are noisiest. The single element is not perfectly flat — a real gradient capsule has its own mild low-frequency self-noise rise (a few dB) and a somewhat higher base EIN than an omni. That rise scales with how much gradient the pattern uses: largest for a pure figure-8, smaller for a cardioid (its pressure component dominates at low frequency, reverting toward omni), and essentially none for an omni. It is far smaller than the array's penalty and has none of the matching/aliasing problems, so for quiet, low-frequency-faithful capture the single directional element is still dramatically cleaner. (The simulator draws this as an illustrative slope, not a measured capsule curve.)

5 · Capsule matching / ATF mismatch

The differential array's deep null only exists if the two capsules are matched in gain and phase. A real gain mismatch of εg or phase mismatch of εφ leaves a residual that fills the null:

null floor (dB) ≈ 20·log₁₀( ½·√(εg² + εφ²) )

A modest 0.5 dB / 2° mismatch already caps the null around −25 to −30 dB no matter how good the design — and it drifts with temperature and aging, and worsens with order. The single element has no matching requirement: its null is set by the acoustic design, not by pairing two parts. One capsule is also cheaper and simpler to manufacture.

6 · What the two-mic array buys you

The array isn't worse at everything — it earns its place through electronic flexibility the fixed element can't offer:

  • Reconfigurable pattern — switch omni ↔ cardioid ↔ hypercardioid ↔ figure-8 in software.
  • Steerable / adaptive null — place a deep null on a moving interferer (the basis of adaptive differential / MVDR).
  • Path to higher order — add mics for narrower beams (more DI) than any first-order element.
  • Tiny footprint — frequency-independent first-order pattern from millimetre spacing.

7 · When to use which

If you need…Choose
Lowest self-noise, best low-frequency SNR, a fixed patternSingle directional element
Robustness, no matching/calibration, lowest cost & powerSingle directional element
Software-switchable pattern, steering, adapting to interferersTwo-mic differential
Higher directivity than first orderHigher-order array (3+ mics)

Rule of thumb: for a fixed first-order pattern, prefer the single directional element — you get the same directivity with far less noise and no matching. Reach for the differential array only when you genuinely need its electronic flexibility, and budget for the low-frequency self-noise and the matched-capsule requirement.

8 · Model & honesty notes

  • First-order, plane-wave, small-spacing (kd ≲ 1) differential model; c = 343 m/s. The closed-form WNG is the standard low-kd approximation (where differential arrays actually operate).
  • The single element is modelled with WNG ≈ 0 dB to represent “no two-channel electronic differencing and no inter-capsule mismatch.” A real pressure-gradient capsule also forms its pattern from a gradient and has its own low-frequency behaviour and proximity effect — but it avoids the dominant penalties modelled here.
  • DI is the diffuse-field directivity; WNG is the spatially-white (sensor-noise) array gain — two different “gains,” which is the whole point.

1 · 比較するもの

一次パターン B(θ) = α + (1−α)·cosθ（カーディオイド、ハイパーカーディオイド等）の指向性マイクを得る2つの方法：

  • 単一指向性素子 — パターンを音響的に形成する1個のカプセル（ポート/迷路を持つ圧力勾配カプセル）。指向性を作る電子回路はない。
  • 2マイク差動アレイ — 小さな間隔 d で離した2個の無指向カプセルを（遅延/等化を伴い）減算し、同じパターンを電子的に合成する。

両者は同一の極座標パターンを作れます。重要なのは各々のコストで、そこで大きく分かれます。

2 · 指向性 — ほぼ引き分け

一次パターンの指向性指数（拡散/等方性雑音に対する利得）は閉形式で表せます：

DI = 1 / ( α² + (1−α)²/3 ) → カーディオイド 4.8 dB · スーパー ≈5.7 dB · ハイパー 6.0 dB

両者は同じパターンを実現するので DI は同じです。一次設計（単一素子でも2マイクでも）は約 6 dB で頭打ち。それを超えるには高次アレイ（3マイク以上）が必要で、以下の問題はすべて悪化します。

3 · 白色雑音利得 — 核心

白色雑音利得（WNG）は、無相関な雑音（センサ自己雑音、電子雑音、風の乱流）に対するアレイの利得です。各チャンネルで独立な雑音を処理がどれだけ増幅するかを示します。

  • 単一素子： 1センサなので増幅するものがない。WNG ≈ 0 dB、周波数に対して平坦。
  • 差動アレイ： 低周波では近接した2マイクがほぼ同じ信号を受け、差は微小（∝ kd = ωd/c）。オンアクシス応答を1に戻すため約 1/(kd) を掛け、無相関雑音も同時に増幅。よって WNG は低周波に向けて崖のように低下（≈ +6 dB/oct）し、間隔が小さいほど悪化します：
WNG ≈ 2 / ( α² + 4(1−α)²/(kd)² ) （小間隔・一次）

具体例（カーディオイド、d = 20 mm）：WNG ≈ 2 kHz で −0.2 dB、1 kHz で −5.9 dB、200 Hz で −19.7 dB、100 Hz で −25.7 dB。間隔を半分にするとさらに約 6 dB 悪化。これが最も重要な実用上の違いです — 周波数と間隔のスライダーを動かし、WNG対周波数のグラフを見てください。

有効範囲： これは小間隔近似で、kd ≲ 1（≈ c/2πd、20 mm で約 2.7 kHz 未満）でのみ成立します。それ以上ではマイク間隔が半波長に近づき空間エイリアシングが起き、WNG は上昇し続けられず、2素子アレイは約 +3 dB の2センサ上限を超えられません。シミュレータは厳密で有界な2素子モデルを用い、高域の曲線を物理的に保ちます。

4 · センサ自己雑音フロア

WNG はそのまま可聴ヒスになります。実効出力自己雑音は、カプセル雑音（EIN）から WNG（dB）を引いたものです：

出力自己雑音 (dB) = EIN − WNG_dB

アレイの WNG は負なので、雑音フロアは上昇します。28 dB(A) のカプセル・20 mm 間隔では、差動カーディオイドの実効自己雑音は 1 kHz で ≈ 34 dB(A) ですが、200 Hz で ≈ 48 dB(A)、100 Hz で ≈ 54 dB(A) — 部屋が最も騒がしい帯域で 20 dB 以上の余分なヒス。単一素子は完全に平坦ではありません — 実際のグラジエントカプセルも低周波で自己雑音がわずかに（数 dB）上昇し、無指向より基準 EIN がやや高めです。この上昇はパターンの勾配成分に比例します：純粋な双指向（フィギュア8）で最大、カーディオイドでは小（低周波では圧力成分が支配し無指向寄りになる）、無指向ではほぼゼロ。いずれもアレイのペナルティよりはるかに小さく、整合やエイリアシングの問題もありません。静かで低域に忠実な収音には、それでも単一指向性素子が圧倒的にクリーンです。（シミュレータではこれを説明用のスロープとして描画しており、実測カプセル曲線ではありません。）

5 · カプセル整合 / ATF 不整合

差動アレイの深いヌルは、2カプセルがゲインと位相で整合している場合にのみ存在します。実際のゲイン不整合 εg や位相不整合 εφ は残差を残し、ヌルを埋めます：

ヌルフロア (dB) ≈ 20·log₁₀( ½·√(εg² + εφ²) )

わずか 0.5 dB / 2° の不整合でも、設計がどれだけ良くてもヌルは約 −25〜−30 dB に制限され、温度・経年で変動し、次数とともに悪化します。単一素子に整合要件はありません：ヌルは2部品のペアリングではなく音響設計で決まります。1カプセルは製造も安価で簡単です。

6 · 2マイクアレイで得られるもの

アレイがすべてで劣るわけではありません。固定素子にはできない電子的柔軟性で価値を発揮します：

  • 再構成可能なパターン — ソフトで無指向 ↔ カーディオイド ↔ ハイパー ↔ 双指向を切替。
  • ステア/適応ヌル — 移動する干渉源へ深いヌルを向ける（適応差動 / MVDR の基礎）。
  • 高次への道 — マイクを足してどんな一次素子より鋭いビーム（高 DI）。
  • 極小サイズ — ミリ間隔から周波数非依存の一次パターン。

7 · 使い分け

必要なもの…選択
最低の自己雑音、最良の低域 SNR、固定パターン単一指向性素子
頑健性、整合/校正不要、最低コスト・低消費電力単一指向性素子
ソフト切替パターン、ステアリング、干渉源への適応2マイク差動
一次を超える指向性高次アレイ（3マイク以上）

目安： 固定の一次パターンなら単一指向性素子を選ぶ — 同じ指向性をはるかに少ない雑音と整合不要で得られます。差動アレイは電子的柔軟性が本当に必要なときだけ採用し、低域の自己雑音と整合カプセル要件を見込んでおきましょう。

8 · モデルと正直な注記

  • 一次・平面波・小間隔（kd ≲ 1）の差動モデル、c = 343 m/s。閉形式 WNG は差動アレイが実際に動作する低 kd の標準近似です。
  • 単一素子は「2チャンネルの電子差分とカプセル間不整合がない」ことを表すため WNG ≈ 0 dB でモデル化。実際の圧力勾配カプセルも勾配でパターンを作り、独自の低域特性や近接効果を持ちますが、ここでモデル化した主要なペナルティは回避します。
  • DI は拡散界の指向性、WNG は空間的に白色（センサ雑音）のアレイ利得 — この2つの「利得」が異なることがまさに要点です。
